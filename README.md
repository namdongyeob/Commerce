# 🛒 Commerce - 실시간 커머스 플랫폼

Java로 구현한 콘솔 기반 쇼핑 플랫폼입니다.  
카테고리별 상품 조회, 장바구니 담기/주문, 관리자 모드를 지원합니다.

---

## 📌 과제 소개

콘솔 환경에서 동작하는 커머스 시스템을 객체지향 설계로 구현했습니다.  
사용자는 상품을 카테고리별로 탐색하고 장바구니에 담아 주문할 수 있으며,  
관리자는 비밀번호 입력 후 상품을 추가·수정·삭제할 수 있습니다.

---

## ✨ 주요 기능

| 구분 | 기능 |
|------|------|
| 사용자 | 카테고리별 상품 조회, 장바구니 담기, 주문 확정, 주문 취소 |
| 관리자 | 상품 추가 / 수정 / 삭제 / 전체 현황 조회 |

---

## ⚠️ 예외 처리

1. **재고 부족 처리** : 재고가 0인 상품은 장바구니에 추가되지 않고 안내 메시지 출력
2. **관리자 비밀번호 3회 오류 처리** : 비밀번호를 3회 틀리면 자동으로 메인 메뉴로 복귀

---

## 🗂️ 클래스 구성

main / CommerceSystem / Category / Product
Cart / CartItem / CartHandler / AdminHandler / Customer

---

## 🗂️ 클래스 구성요소

| 클래스 | 역할 |
|--------|------|
| Main | 프로그램 시작점 |
| CommerceSystem | 메인 메뉴 출력 및 흐름 관리 |
| Category | 카테고리별 상품 목록 관리 |
| Product | 상품 정보 (이름, 가격, 설명, 재고) |
| Cart | 장바구니 전체 관리 |
| CartItem | 장바구니 항목 하나 (상품 + 수량) |
| CartHandler | 장바구니 담기/주문/취소 처리 |
| AdminHandler | 관리자 인증 및 상품 추가/수정/삭제 |
| Customer | 고객 정보 (이름, 이메일, 등급) |
